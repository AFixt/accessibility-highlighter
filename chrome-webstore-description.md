# Accessibility Highlighter - Chrome Web Store Description

## Short Description (132 characters max)

Instantly visualize accessibility barriers on any webpage with color-coded highlights and accessible console reports.

## Detailed Description

### What It Does

The Accessibility Highlighter makes accessibility problems visible. With one click, see exactly where barriers exist on any webpage through intuitive color-coded overlays. Perfect for developers, designers, and anyone who wants to understand what makes websites inaccessible.

### Key Features

#### Instant Visual Problem Discovery

- Color-coded overlays show exactly where accessibility issues exist
- Diagonal stripe patterns indicate different types of problems
- Each overlay displays the specific accessibility barrier
- One-click toggle reveals the hidden accessibility landscape

#### Comprehensive Accessibility Detection

- Missing or inadequate alt text on images
- Form elements without proper labels
- Buttons and links without accessible names
- Tables missing headers or captions
- Iframes without descriptive titles
- Poor heading structure and hierarchy
- Uninformative or generic link text
- Color contrast problems
- Missing ARIA attributes and landmarks

#### Accessible Console Reporting

- Structured console logs provide accessible alternative to visual overlays
- Screen reader compatible issue descriptions
- Detailed explanations for users who cannot see visual highlights
- Programmatic access to accessibility data for assistive technologies

#### Seamless Integration

- Keyboard shortcuts for quick activation (Ctrl+Shift+A)
- Lightweight and fast - no page performance impact
- Works on all websites and web applications
- Remembers your preferences across browser sessions

### How It Works

1. **Click the extension icon** to reveal accessibility barriers
2. **See instant visual overlay** highlighting problem areas with colors and patterns
3. **Access console reports** for screen reader users and detailed technical information
4. **Navigate with keyboard shortcuts** to explore detected barriers
5. **Understand accessibility impact** through clear, descriptive explanations

- **Zero Performance Impact**: Lightweight analysis engine that doesn't slow down page loading

### Getting Started

1. Install the extension from the Chrome Web Store
2. Navigate to any webpage
3. Click the Accessibility Highlighter icon in your toolbar
4. Instantly see accessibility barriers highlighted with colored overlays
5. Open Developer Tools Console (F12) to access screen reader compatible reports
6. Use Ctrl+Shift+A (Cmd+Shift+A on Mac) for quick toggling
